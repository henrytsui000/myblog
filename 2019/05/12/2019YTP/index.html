2019YTP

Author: oxolane Date: 五月 12, 2019  0:00:00

一大早就到清大客運站和王師宇電神圈會合
然後一起搭客運上台北
中間聽蘇柏瑄講跟潘老師一起去比賽的事
小六就會三角函數跟算衛星週期 真的有夠電

到台北之後決定走到賽場
中間遇到原本以為會很盛大的遊行
結果超費的QAQ 爾且他們頂尖大學的名單裡面沒有MIT耶
學姊情何以堪

到賽場之後發現是按照隊長所屬縣市排座位
新竹區一大坨在前面 我是高雄區XD
到位置之後看到翁菀羚正在寫code跟快速debug好電喔
然後李家杰在解釋如何獲得超新的無限大 猛
比賽開始
時間其實沒有很確定 都是推估的
[00:00]比賽開始
[00:01]發現p4凱薩加密水題 沒種的我給隊友
[00:07]p4好像有BUG 翁菀羚把送分題解決了
[00:11]p4也被解決 我在亂翻題目 李家杰在家庭代工
[00:15]開始想做p1 但是題目有點冗
[00:20]發現 陳庠宇xd
[00:21]發現看錯p1 人生困難
[00:49]決定好好做p1不要分心
[01:10]因為太懶 直接弄出結論公式 (對啦我是很笨的毒瘤)感謝隊友的四捨五入函式 AC
浪費時間階段:
發現p5是數字龍捲風 可是很討厭處理格式
跟隊友誤導卡車是0/1背包問題而已
再次誤導隊友史萊姆是史萊姆問題
試著生出競賽分組的直接公式
結果後來聽bb說只要二元樹模擬下去就好了QQ
[03:00]準備弄沒人要的p2
[03:20]我不會看題目
[03:25]一筆測資TLE好難過喔
[03:35]翁菀羚通靈出測資 AC 根本天才
小插曲:原本想把費式數列用矩陣快速冪優化 結果不小心講成O(1)費式數列了QQ不是故意要玩梗的
廢到比賽結束

晚餐走道京站進食
被一大群實科的電神包圍好榮幸喔
然後吃到一半看到有人穿竹女制服路過???什麼鬼
總之晚餐蔬菜好多 南瓜難吃 店員根本千里耳

心得
1.我好雷我道歉
2.我好愛裝熟QQ
3.要學會轉2*2魔方 這攸關到20分
4.住在台北醫院時間比住家裡多是合理的
5.FB會偷吃文章 組刻薄拜託不要ban我
6.5[arr]是合法的
7.其實可以寫出沒有分號的C++ code
8.崔浩堂又再發流水帳囉
