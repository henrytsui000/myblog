2019IONCamp

Author: oxolane Date: 八月 3, 2019  0:00:00

<國培旁聽+IONCamp>
原本想要昨天發，順便祝ISSC大家順利
結果不小心睡著了 嗚嗚嗚嗚

國培旁聽

7/16

跟朋友在二餐吃午餐的時候巧遇資奧國手們
然後BB就來問之後要不要來旁聽www
隔天也沒想那麼多就去了

7/17

中午跟 王師宇 波路特神在二餐吃滷肉飯
被滷肉飯價格跟好吃程度嚇到 發現綠洲><
下午開始上課 因為對我來說超困難
所以只有大概聽懂nash equilibrium跟stable match
之後大腦就不省人事了XD
下課之後大家留下來討論一陣子
不小心提到我的會考國文慘案
然後就被拿來跟競程比較了
比較詳情請見 蘇柏瑄 （Brian Su） 的某天日記XDD
7/18
早上決定還是過去被電 能吸收多少算多少
上課內容是greedy 但是記憶僅剩matroid和其他一點點
下午決定去建功補考中華文化基本教材
因為蠻想旁聽的 20分鐘就提早交卷跑了
回到教室之後發現錯過排容原理 so sad
接著是線性規劃的部分 好難 輸光光
下課後我跟著電神們跑到宿舍(?)
順便聽BB講解為什麼要用sublime
耶 終於脫離devC

晚上跟很久很久以前的WMTC隊友聚餐
大家都好電 尤其是曹偉要發射火箭
被電爛ㄌ

7/19

早上繼續去國培旁聽，內容是隨機增量法
雖然沒有完全吸收 不過概念大概是懂ㄌ
然後才知道一些喇分數其實不少見(日後再IONC體驗到了)
中午看到 唐博彥 在空閒時間做科展 超電(X
下午教授播MIT的上課影片
因為英文能力低落 又再加上內容對我來說頗難
所以就睡倒了 教授對不起QQQQ
跟國手們座車到高鐵站的路上
Brian Tsai 說我可以跟大家宣稱我又上過國培課程
聽起來會變很電 超有道理(X
<旁聽國培結束>

再來是IONC流水帳

IOCamp

day1

中午報到的時候剛好遇到 陳冠辰 超幸運
害怕因為沒有認識的在營隊很邊緣@@
下午在講排序等 Aann Lin 直接大爆睡 超電
晚上模擬賽 坐在 翁菀羚 和 陳冠辰 旁邊
壓力超大><
最後東東電爛大家 <(_ _)>
我超廢 也就不講題目了
晚上跟室友打橋 打到三點半才睡www

day2

早上用盡力氣爬到教室後
發現課程是排序 就繼續補眠了
中午是不知啥午餐搭配2倍速成龍歷險記
然後終於看到了 Allen Guan 林哲宇
也成功加到 黃雅筠 FB(嗚嗚嗚申請半年被無視
下午課程是DP 因為以為自己很電就去進階班了
但我是廢物 只能剛好壓線吸收
下午茶時間聽電電東講N平方史萊姆 好電
晚上上機一樣廢 然後被 翁菀羚 排擠嗚嗚嗚
回宿舍後因為沒有記取教訓
又跟室友們打牌到深夜XXD

day3

早上一樣是困難的爬起來
然後上資結 因為資結的部分很廢
所以其實學到很多想學的
包括從IOIC就想學的持久化線段樹 等等
中午仍然是附近的便當店搭配成龍X2
下午是圖論 因為腦子清醒了
就保持正常吸收
不過化了很久的時間在看金科拉XDD

下午茶時間
才知道 全都是高一升高二
只有我是臭老人 所以就開始宣稱我要重考國中
之後的幾天就被他們輪流叫學弟跟老人了QQ

day4

今天都是 葉柏宏 的課
早上是trie 我都念tree 才發現原來是try喔
因為再度被 排擠 我就跑到電電東旁邊吸收電神經華
發現他真的好電<(_ _)>
下午的課是數學
因為很想聽FFT，所以就跑去請講師教學www
然後FFT真的好難 然後有人說可以直接打出來 不用背 電爛
晚上要走到電腦教室的時候因為一個人走
黃雅筠 就過來跟我說”欸你真的沒朋友耶”
心理傷害-INF
回宿舍後因為隔天是最後一天了
就跟室友通宵打牌
在3點多的時候想不開 決定玩心臟病
直接輸到崩潰 還引來 Allen Guan XDD
5點多的時候因為大家都很累
所以提議跳舞 然後就這樣瘋瘋癲癲的通霄了

day5

早上是greedy 發現附近不少電神睡著了
所以也想裝電的跟著睡(X
然後就跳過早上課程了@@
下午是正式考試
因為糾結在邊長1*1是不是矩形
時間噴光光 所以成績很爛
嗚嗚嗚也就不提考試過程了
恭喜陳冠辰第一 林哲宇第三
我一開始就傳一個空白程式碼
結果居然沒有搶到首WA 好難過喔

最後
祝國手們IOI比賽順利
全都拿釒~~
謝謝讓我旁聽國培www
